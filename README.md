# Project VueJS Batch 27

## Deskripsi

Repositori ini adalah repositiori yang berisi tugas proyek akhir untuk bootcamp online VueJS (Batch 27) di SanberCode. Proyek ini menggunakan VueJS 2 & Vue CLI.
